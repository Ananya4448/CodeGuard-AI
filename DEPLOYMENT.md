## Next Steps

### 1. View Your Repository

Visit: https://github.com/Ananya4448/CodeGuard-AI

### 2. Clone the Repository (from another location)

```bash
git clone https://github.com/Ananya4448/CodeGuard-AI.git
cd CodeGuard-AI
```

### 3. Set Up Development Environment

```bash
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
```

### 4. Configure Environment

```bash
cp .env.example .env
# Edit .env and add your OPENAI_API_KEY
```

### 5. Run the Project

```bash
# Run basic example
python examples/basic_usage.py

# Or start the API server
python -m src.api.server
```
